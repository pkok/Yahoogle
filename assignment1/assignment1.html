Assignment 1

      • Goal

    The goal of the first assignment is to get acquainted with an IR system and to make sure you acquire the skills you need for future IR experiments. The assignment has 4 important steps: (1) install an IR system, (2) index a document collection, (3) run a set of queries on the index, and (4) evaluate the results. You are expected to hand in a report (max. 4 pages) describing what you did and what the results are of your actions.

    After finishing this assignment you should be able to index collections using an IR system, run a set of queries on the index, and evaluate the results. You need these skills for the next assignment!

        Grade

    The grade will be determined by the completeness of your report (we must be able to replicate what you did) and by the correctness of the experiment.

        Submitting

    You can work in teams of up to 2 people on this assignment, but make sure you put the names and student IDs of all team members in the report. Both team members need to submit the (same) report on Blackboard, so that we can easily see who didn't submit. Deadline: November 11, 2012 before 23:59

        Step 1

    - Install Indri (http://www.lemurproject.com) or another IR system (e.g., Lucene or Terrier).
    - Although most IR systems have a GUI (graphical interface), it is usually easier to run experiments using the command line interface.
    - For Indri we have a small tutorial available explaining how to use it.

        Step 2

    - Download two document sets (GH95.tar.gz and LAT94.tar.gz) made available at http://staff.science.uva.nl/~mdr/Teaching/IRTestMaterials/  (username: ir ; password: 0910). Get both, you need both. These are English language news papers used at earlier editions of CLEF. Do not redistribute this material. Store it in a directory that is only readable by you.
    - Unzip, untar (use 7-zip for this on Windows machines), and take a look at one of the files so that you know what the format is, and do the "one document task" with this file (i.e., index it).
    - If you succeed in the one-document task, proceed to indexing the whole collection. Make sure to record the choices you make for the report.

        Step 3

    - Download the CLEF 2006 topics. For Indri and Lemur we have preprocessed topics, for other systems you might need to preprocess the original topics for the IR system to be able to read them.
    - Run the set of topics on your index (step 2) and store the results.

        Step 4

    - Download the evaluation script (trec_eval) and relevance assessments.
    - Run the script on the results of step 3 using the relevance assessments. Check if the numbers make sense (all 0.000 is not good;) ) and report on the most important metrics (MAP, P10, ...)

        Step 5

    - Write a (max 5-page) report about this assignment
    - Show the important information (e.g., hoe did you check that your index is ok? Which settings did you use for indexing? Which setting did you use for retrieval? What are the scores? Etc.)
