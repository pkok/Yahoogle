Assignment 3

      • Goal

    The goal of this assignment is to get creative, in a scientific way. Assignment 2 was aimed at designing a small scale experiment, in this assignment we take this one step further. During the course you've heard and read about IR-related research and its many applications. Given the many possibilities in the field of IR, we ask you to design a novel IR-related experiment, without actually implementing it.

    In many ways, this assignment is similar to assignment 2, but since you do not have to implement the actual system or perform the experiments, we expect you to come up with novel and interesting research questions, experimental setup, and analyses. The final report for this assignment (max. 7 pages) should exist of the same sections as assignment 2, except for the results and conclusions sections. See the example for more details.

    After finishing this assignment you should be able to set up your own IR-related research/ experiment, capable of reviewing relevant literature, and know how specific research questions should be evaluated and which metrics and analyses are good for this.

        Grade

    The grade will be determined based on how innovative the experiment is and whether (a) related work is complete, (b) the metrics are appropriate, (c) evaluation is properly described, (d) analyses are sufficient, and whether the complete experiment makes sense.

        Submitting

    You can work in teams of up to 2 people on this assignment, but make sure you put the names and student IDs of all team members in the report. Both team members need to submit the (same) report on Blackboard, so that we can easily see who didn't submit. Deadline: December 23, 2012 before 23:59

        Example

    • I want to find out if I can use social media to predict movie ratings

    • RQ1: "Can we predict movie ratings using textual information from Twitter?"

    • RQ2: "Can we use Youtube view count and comments to predict ratings?"

    • Hypothesis: I expect Youtube features to be most useful, leading to better results than Twitter.

    • Related work: other prediction tasks using social media.

    • Model: we train a regression model on a set of 10 movies, collected from IMDb. We predict the actual rating (1-10) using a set of features. The features are the following: number of tweets, number of Youtube views, number of comments, ratio like/dislikes, unigrams in tweets, unigrams in comments, ...

    • Data: we get a set of 60 very popular movies and their ratings from IMDb. Using each of these movies, we collect tweets for a period of 4 weeks and monitor Youtube activity as well.

    • Experiment: train a model on 10 movies, apply the trained model to 50 movies. We try the following three feature set: Twitter-only, Youtube-only, All.

    • Analysis: we report Mean Average Error and compare the three feature sets on this metric. Additionally, we explore the predictions for individual movies and zoom in on movies their show very bad or very good performance and try to explain their behavior.
