Assignment 2

      • Goal

    The goal of this assignment is to run a small scale IR experiment. You should be able to index the collection and run queries without problems now, so the focus is more on the experimental design and the evaluation of the results. More specifically, you are requested to write a report in which you propose a (small) IR experiment, including related work and hypotheses, perform the experiment, and evaluate and analyze the outcomes of the experiment. The report should be at most 7 pages and should contain the following elements:
    - Introduction, including research questions and hypotheses
    - Some related work
    - Description of the data (collection), preprocessing, and (indexing/retrieval) methods
    - Evaluation results
    - Analysis of the results (plots, comparisons, per-topic analysis)
    - Conclusions

    After finishing this assignment you should be able to design a (small) IR experiment and have a feeling for how to evaluate the outcomes.

        Data

    For this experiment we have the following data available. Using this data we expect you to come up with a good experimental design.

    • Document collections (for more info, check CSIRO-collection.pdf). The collections are already processed and ready to be indexed (trectext format). We distribute four different versions of the collection:

      • corpus.body: the body of the web pages (160MB)

      • corpus.title: the text in the title-tag of the web pages (3MB)

      • corpus.metadata: the information in the meta tags (description and keywords) (3MB)

      • corpus.headers: the text between html heading tags (h1 .. h6) (9MB)

    • Topics. TREC Enterprise 2007 has 50 topics, numbered 1--50. We give you the Indri queries and the original queries.

    • Relevance assessments (qrels).

        Grade

    The grade will be determined based on the completeness of the report, the difficulty of experiments and/or analyses, and (of course) whether the design and outcomes make sense.

        Submitting

    You can work in teams of up to 2 people on this assignment, but make sure you put the names and student IDs of all team members in the report. Both team members need to submit the (same) report on Blackboard, so that we can easily see who didn't submit. Deadline: December 2, 2012 before 23:59

        Example

    I want to compare retrieval performance on two indexes: Index A is created without stopword removal and Index B with stopword removal. The research questions are "Does stopword removal improve retrieval performance?" and "Do all topics show the same performance change or do we find per-topic differences?". I hypothesize that stopword removal helps recall, but harms precision, and that we only find performance changes for topics that actually have stopwords in the query.

    • Related work

    • Description of data and stopword list; retrieval model (tf-idf for both indexes)

    • Results (MAP, recall@1000, precision@10)

    • Analysis: compare the results of the two runs for the metrics you are interested in. Compare results on a per-topic basis. Which topics show interesting behavior?

    • Conclusions: what are the answers to my questions? Were my hypotheses right or not, and why (not)?

    (You can re-use this example, but your grade will never be higher than a 7, even if executed perfectly).
